Room 1: Class Blueprint

Choose the class that matches the spec

A junior developer tried to implement a Gadget class, but several versions ended up in the codebase. Only one version correctly follows the project specification below.

Specification: The Gadget class must have two instance variables (name and charge), a constructor that initializes both, and an accessor method that returns the current charge.

Study each option and select the one class that correctly meets the specification.

Option A

public class Gadget {
    private String name;
    private int charge;

    public Gadget(String gadgetName, int startCharge) {
        name = gadgetName;
        charge = startCharge;
    }

    public int getCharge() {
        return charge;
    }
}

Option B

public class Gadget {
    public String name;
    public int charge;

    public Gadget(String gadgetName) {
        name = gadgetName;
        charge = 0;
    }

    public void getCharge() {
        System.out.println(charge);
    }
}

Option C

public class Gadget {
    private String name;
    private int charge;

    public Gadget(String name, int charge) {
        name = name;
        charge = charge;
    }

    public int getCharge(int extra) {
        return charge + extra;
    }
}

Option D

public class gadget {
    private String name;
    private int charge;

    public gadget(String gadgetName, int startCharge) {
        name = gadgetName;
        charge = startCharge;
    }

    public int charge() {
        return charge;
    }
}
