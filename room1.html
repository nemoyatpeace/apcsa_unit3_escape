Room 1: Class Blueprint

Assemble the password from class pieces

The project files are shattered into fragments of Java classes. For each fragment, choose the option that correctly matches the specification.

Each correct choice reveals a letter. Put the letters together (in question order) to form a 5-letter password that will unlock Room 2.

Question 1

Choose the class declaration and fields that best match this spec: “A class for a device, named Gadget, with instance variables name (String) and charge (int).” 

// Which version is best?
<A> public class gadget {
        public String name;
        public int charge;
    }

<L> public class Gadget {
        private String name;
        private int charge;
    }

<Z> private class Gadget {
        private String name;
        private int charge;
    }


Question 2

The Gadget spec continues: “The constructor should take a name and an initial charge and initialize both instance variables.” Which constructor matches that behavior? 

// Assume fields: private String name; private int charge;

<I> public Gadget(String gadgetName, int startCharge) {
        name = gadgetName;
        charge = startCharge;
    }

<B> public Gadget(String name, int charge) {
        name = name;
        charge = charge;
    }

<Q> public Gadget(String gadgetName) {
        name = gadgetName;
        charge = 0;
    }


Question 3

The spec: “Provide an accessor method that returns the current charge as an int.” Which method fits that description? 

<G> public int getCharge() {
        return charge;
    }

<R> public void getCharge() {
        System.out.println(charge);
    }

<M> public int getCharge(int extra) {
        return charge + extra;
    }


Question 4

Now suppose the spec adds: “Provide a mutator that sets the charge to a new value.” Which option is the best match? 

<H> public void setCharge(int newCharge) {
        charge = newCharge;
    }

<S> public int setCharge(int newCharge) {
        return newCharge;
    }

<K> public void charge(int amount) {
        charge += amount;
    }


Question 5

Finally, the spec says: “Add a method that returns a String description containing the name and charge.” Which option aligns best with that idea? 

<T> public String toString() {
        return name + " (" + charge + "%)";
    }

<X> public void printInfo() {
        System.out.println(name + " " + charge);
    }

<P> public String getName() {
        return name;
    }


As you answer, record the letter from each correct choice in order to form a 5-letter password.
